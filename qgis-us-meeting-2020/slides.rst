QGIS Server?
============

+ Map server based on QGIS core library
+ Same release cycle with a **qgis-server** package
+ Numerous services:
    + WMS 1.1.0 and WMS 1.3.0
    + WFS 1.1 and OGC API Features (aka WFS3)
    + WMTS
    + WCS
